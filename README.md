# Possibility Ethics: Real-Data Audit Using the Open University Learning Analytics Dataset (OULAD)

![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)

## Overview

This repository contains the reference implementation of the real-data audit presented in the manuscript:

> **Possibility Ethics: Measuring Moral Value as the Expansion of Safe Future Options in Sociotechnical Systems**

The software demonstrates how the Possibility Ethics framework can be applied to a real educational dataset using the **Open University Learning Analytics Dataset (OULAD)**. It implements the complete computational audit pipeline, including deterministic pathway construction, viability filtering, harm-admissibility assessment, Option Entropy computation, vulnerability-sensitive aggregation, cohort-level non-regression analysis, and Option Repair identification.

The repository is intended to support **reproducible computational research** and to accompany the associated scholarly publication.

# Methodology

The implementation follows the mathematical framework described in the accompanying paper.

The computational workflow consists of:

1. Loading the OULAD dataset.
2. Constructing deterministic student pathway signatures.
3. Applying viability filtering.
4. Applying harm-admissibility constraints.
5. Identifying admissible opportunity classes.
6. Computing Option Entropy

\[
\Omega=\log(\kappa+K)-\log(\kappa)
\]

7. Computing vulnerability-weighted opportunity levels.
8. Detecting cohort-level non-regression.
9. Generating Option Repair recommendations.
10. Producing publication-ready tables and figures.

# Interpretation

This repository demonstrates the application of the Possibility Ethics framework to **real observational educational data**.

The analysis evaluates structural opportunity patterns under explicitly declared assumptions.

It **does not** estimate causal effects, educational interventions, or policy impacts.

The reported values should therefore be interpreted as **computational audit measures** rather than causal estimates.
