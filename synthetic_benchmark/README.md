# Synthetic Benchmark

This directory contains the deterministic synthetic benchmark used in Section 6.1 of the manuscript.

Purpose
- Verify the computational behaviour of the Possibility Ethics framework.
- Reproduce the values reported in Table 3.
- Demonstrate deterministic Option Entropy computation.

Parameters
- τ = 0.60
- γ = 0.20
- θ = 0.75
- κ = 1

The benchmark is illustrative rather than empirical. No random sampling is used. All Option Entropy values, weighted opportunity levels, rankings, and benchmark outputs are deterministic and reproducible with the Omega-PE implementation.

Version: 1.0.1
